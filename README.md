# Corteyo

A modern finance website built with React and Vite.

## 🚀 Live Site

Hosted on GitHub Pages: [https://your-username.github.io/corteyo](https://your-username.github.io/corteyo)

## 📁 Project Structure

```
dist/
├── index.html          # Main entry point
├── favicon.png         # Site favicon
├── assets/
│   ├── index-*.js      # Bundled JavaScript
│   └── index-*.css     # Bundled CSS
└── ...                 # Images and other assets
```

## 🛠️ Tech Stack

- **React** — UI framework
- **Vite** — Build tool
- **DM Sans** — Google Font

## 📦 Deployment (GitHub Pages)

1. Extract the contents of the `dist/` folder
2. Push all files to the `gh-pages` branch (or `main` branch root)
3. Go to **Settings → Pages** in your GitHub repo
4. Set the source to the correct branch
5. Your site will be live in a few minutes
